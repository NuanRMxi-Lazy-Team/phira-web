# phira-web | PhiraMpServerCSharp 修改版

本版本相对于原版 [Phira-Web](https://github.com/teamflos/phira-web)添加了多人联机服务器页面，但是请注意，这个功能仅针对 [PhiraMpServerCSharp](https://github.com/NuanRMxi-Lazy-Team/PhiraMpServerCSharp) 服务器，它不能用于原版服务器，并且你需要部署外部接口HTTP服务器。  
此项目仍在开发，欢迎一同维护，联系我：nrlt@nuanr-mxi.com

## 项目设置

```sh
pnpm install
```

### 编译和热重载用于开发

```sh
pnpm dev
```

### 类型检查、编译和压缩用于生产

```sh
pnpm build
```

### 使用 [ESLint](https://eslint.org/) 进行代码检查

```sh
pnpm lint
```
